- [virtualenv >= 20.14.0]

## Start the server

Create virtual environment (optional)

```
$ virtualenv venv
$ source venv/bin/activate
```

Pull the dependencies and start the server

```
$ pip install -r requirements.txt
$ python app.py
```

*** Open browser, and navigate to localhost:5000
